# 支付通道基础

## 什么是支付通道

支付通道是资金流转的核心链路，连接商户、平台与银行/清算机构。常见类型：

- **直连银行通道**：与银行直接签约，费率低，门槛高
- **第三方支付通道**：通过支付宝、微信支付等接入，适合中小商户
- **聚合支付通道**：一个接口聚合多个通道，灵活切换
- **跨境支付通道**：处理多币种、跨境结算

## 通道选择核心指标

| 指标 | 说明 |
|------|------|
| 费率 | 单笔交易手续费，通常 0.1%~1% |
| 稳定性 | 通道可用率，核心指标 |
| 到账速度 | T+0 / T+1 / D+1 |
| 限额 | 单笔/日累计限额 |
| 合规资质 | 是否持牌，是否在监管范围内 |

## 通道对接前必须确认

1. 对方是否持有支付业务许可证
2. 结算周期与提现规则
3. 异常交易处理流程（拒付/退款/争议）
4. 数据安全与加密标准
5. 沙盒环境是否完整
