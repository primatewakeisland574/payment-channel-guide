# 常见问题与踩坑记录

## Q1：回调一直收不到怎么办？

1. 确认回调 URL 外网可访问
2. 确认服务器 IP 已加入通道白名单
3. 检查 HTTPS 证书是否有效
4. 查看通道后台是否有重试记录
5. 用 curl 模拟回调测试本地接口是否正常响应

## Q2：订单状态一直是「待支付」没有更新？

- 回调未收到（见 Q1）
- 回调处理逻辑报错，未更新数据库
- 未正确返回成功标识，通道认为失败未推进状态

## Q3：签名验证失败？

- 检查密钥是否正确（注意有无空格/换行）
- 检查参与签名的字段是否与文档一致
- 金额单位确认（元还是分）
- 编码统一为 UTF-8

## Q4：提现被拒/结算延迟？

- 确认账户实名认证状态
- 检查是否触发风控（大额、频繁提现）
- 联系通道客服获取具体原因

## Q5：如何选择稳定的支付通道？

- 优先选择持牌机构
- 参考同行口碑，小额先测试稳定性
- 多备几条通道，避免单点故障
- 签合同前确认结算周期与异常处理流程

## Q6：对账文件格式不标准怎么处理？

- 提前与通道确认文件格式（CSV/Excel/API）
- 写通用解析脚本，做字段映射
- 保留原始文件，处理结果单独存储
