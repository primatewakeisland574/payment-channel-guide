# 风控规则与合规

## 常见风控维度

| 维度 | 说明 |
|------|------|
| 交易频率 | 单用户/单设备单位时间内交易次数 |
| 金额异常 | 单笔金额超阈值或连续小额拆单 |
| 地理位置 | IP 归属地与常用地不符 |
| 设备指纹 | 同设备多账号，或同账号多设备 |
| 黑名单 | 历史风险用户/商户 |

## 拒付（Chargeback）处理

- 建立完整的交易记录留存（至少 5 年）
- 及时响应通道的争议通知（通常 7~15 天）
- 保留用户授权证明、物流信息等证据

## 合规要点

- 不得为无牌机构做资金通道
- 用户实名认证（KYC）
- 大额交易上报（国内单笔 5 万以上需上报）
- 反洗钱（AML）监控
